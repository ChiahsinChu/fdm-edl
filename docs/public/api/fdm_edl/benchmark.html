Analytical GCS solution for a symmetric binary (z:z) electrolyte.

The electrode potential \( \phi_0 \) is connected to the OHP potential \( \phi_d \) through the Stern capacitor:

$$\sigma = \frac{\varepsilon_s}{d}(\phi_0 - \phi_d)$$

and also to the diffuse-layer charge via the Grahame relation:

$$\sigma = \sqrt{8\,\varepsilon\,k_BT\,c_0\,N_A}\; \sinh!\left(\frac{z\,e\,\phi_d}{2\,k_BT}\right)$$

Equating these gives an implicit equation for \( \phi_d \), which is solved numerically (Brent). The diffuse-layer potential profile is then the non-linear PB closed form:

$$\phi(x) = \frac{4\,k_BT}{ze}\,\mathrm{arctanh}!\left[ \tanh!\left(\frac{ze\,\phi_d}{4\,k_BT}\right) \exp!\left(-\frac{x}{\lambda_D}\right)\right]$$

  • d_ohp (unxt.Quantity, optional): Distance from electrode surface to the outer Helmholtz plane (default: 5 Å).
  • eps_ohp (float, optional): Relative permittivity inside the Stern (compact) layer (default: 6.0).

class NonLinearPoissonBoltzmann(fdm_edl.benchmark.pb1d.BasePoissonBoltzmann): 
153class NonLinearPoissonBoltzmann(BasePoissonBoltzmann):
154    """Analytical non-linear Poisson-Boltzmann profile in 1D.
155
156    Users provide an ElectricalDoubleLayer object and can optionally provide
157    custom spatial coordinates and surface charge density.
158
159    Notes
160    -----
161    This closed-form solution is valid for a symmetric binary (1:1)
162    electrolyte only. It is not valid for mixed-valence or multi-component
163    electrolytes.
164    """
165
166    def __init__(
167        self,
168        edl_obj: ElectricalDoubleLayer,
169    ):
170        super().__init__(edl_obj=edl_obj)
171        self.bulk_concentration, self.valency = (
172            self._validate_symmetric_binary_electrolyte()
173        )
174
175    def _validate_symmetric_binary_electrolyte(
176        self,
177    ) -> tuple[unxt.Quantity, unxt.Quantity]:
178        ions = self.edl_obj.electrolyte.ions
179        if len(ions) != 2:
180            raise ValueError(
181                "NonLinearPoissonBoltzmann analytical solution requires exactly "
182                f"2 ions (symmetric z:z electrolyte), got {len(ions)}."
183            )
184
185        z_tot = 0.0
186        zi = 0.0
187        for _name, ion in ions.items():
188            zi = float(
189                ion.charge.to("C").value / constants.ELEMENTARY_CHARGE.to("C").value
190            )
191            z_tot += zi
192
193        # symmetric binary electrolyte: valences must be +1 and -1
194        if not (jnp.isclose(z_tot, 0.0)):
195            raise ValueError(
196                "NonLinearPoissonBoltzmann analytical solution requires a "
197                "symmetric z:z electrolyte."
198            )
199
200        c0 = ion.molar_conc.to("mol/L")
201
202        return c0, unxt.Quantity(
203            jnp.abs(zi), unit=cds.e
204        )  # return bulk concentration and valence magnitude
205
206    def phi0_to_sigma(
207        self,
208        phi_0: unxt.Quantity,
209    ) -> unxt.Quantity:
210        """Grahame equation: sigma = sqrt(8 eps k_B T c_0 N_A) sinh(z e phi_0 / 2 k_B T)."""
211        epsilon = self.edl_obj.electrolyte.solvent.eps_0 * constants.VACUUM_PERMITTIVITY
212        grahame_prefactor = jnp.sqrt(
213            8
214            * constants.BOLTZMANN_CONSTANT
215            * self.edl_obj.temperature
216            * epsilon
217            * self.bulk_concentration
218            * constants.AVOGADRO_NUMBER
219        )
220        arg = (self.valency * self.beta * phi_0 / 2).to("")
221        return (grahame_prefactor * jnp.sinh(arg)).to(cds.e / unxt.unit("angstrom^2"))
222
223    @staticmethod
224    @jax.jit
225    def compute_phi(x, phi_0, debye_length, beta, valency):
226        gamma = jnp.tanh(beta * valency * phi_0 / 4)
